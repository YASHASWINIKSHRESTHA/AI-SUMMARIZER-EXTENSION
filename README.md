
# 🧠 AI Article Summarizer Chrome Extension

## 📋 Overview
**AI Article Summarizer** is a Chrome extension that uses AI to quickly summarize long articles or webpages into concise, easy-to-read summaries.  
Simply click the extension icon, and it extracts the main content from the current webpage and provides a short AI-generated summary — saving time and boosting productivity.

---

## ⚙️ Features
- 📰 **Instant Summaries** — Get short, clear summaries of any webpage.  
- 🤖 **AI-Powered** — Uses advanced language models to understand and summarize content intelligently.  
- ⚡ **Lightweight & Fast** — Runs smoothly without slowing down your browser.  
- 🧩 **Simple UI** — Clean popup design for quick use.  
- 🔒 **Privacy-Focused** — Processes only visible article text, without storing your personal data.

## 🚀 How to Use

1. **Clone this repository**
   ```bash
   git clone https://github.com/YASHASWINIKSHRESTHA/AI-SUMMARIZER-EXTENSION.git
````

2. **Open Chrome Extensions Page**

   * Go to `chrome://extensions/`
   * Turn on **Developer Mode** (top right)

3. **Load Unpacked Extension**

   * Click **Load unpacked**
   * Select your project folder

4. **Use It**

   * Visit any article or webpage
   * Click the **AI Article Summarizer** icon in the toolbar
   * Get your summary instantly!
